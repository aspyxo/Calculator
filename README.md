#Калькулятор

##О проэкте
- В этом репозитории представлен калькулятор с возможность счить: Базовые действия, Квадратный корень, Корень n-й степени, Степень, Факториал, Логарифм натуральный, Логарифм десятичый, градусы Синуса и Косинуса. Может быть полезен для: Учебы, Работы и отладки, Научных расчетов.

##Технологии
- С++
- Библиотеки (stdio.h, math.h)

##Как запустить
1. Скопировать Код
2. Открыть онлайн компилятор или VS
- Чтобы запустить калькулятор в онлайн компиляторе необходимо: свтавить заранне скопированный код, выбрать язык C++, нажать кнопку Run.
- Чтобы запустить калькулятор а VS необходимо: Создать проэкт (выбрать консольное приложение С++), вставть код в программу, нажать комбинацию клавишь ctrn + F5.

#In English

#Calculator

##About the project
- This repository contains a calculator with the ability to calculate: Basic operations, Square root, Nth root, Power, Factorial, Natural logarithm, Decimal logarithm, Sine and Cosine. It can be useful for: Studying, Working and debugging, Scientific calculations.

##Technologies
- С++
- libraries (stdio.h, math.h)

##How to launch
1. Copy Code
2. Open the online compiler or VS
- To run the calculator in the online compiler, you need to: paste the previously copied code, select the C++ language, and click the Run button.
- To launch the calculator in VS, you need to: Create a project (select a C++ console application), paste the code into the program, and press the key combination ctrn + F5.
