#Калькулятор

##В этом репозитории представлен калькулятор с возможность счить: Базовые действия, Квадратный корень, Корень n-й степени, Степень, Факториал, Логарифм натуральный, Логарифм десятичый, градусы Синуса и Косинуса. Может быть полезен для: Учебы, Работы и отладки, Научных расчетов.

##Технологии
- С++
- library (stdio.h, math.h)

##Как запустить
1. Скопировать Код
2. Открыть онлайн компилятор или VS studio
2.1 Чтобы запустить калькулятор в онлайн компиляторе необходимо: свтавить заранне скопированный код, выбрать язык C++, нажать кнопку Run.
2.2 Чтобы запустить калькулятор а VS studio необходимо: Создать проэкт (выбрать консольное приложение С++), вставть код в программу, дописать в самый вверх "#define _USE_MATH_DEFINES" (без ""), нажатькомбинацию клавишь ctrn + F5. Если будет ошибка scanf небезопасен дописать в начало "#define _CRT_SECURE_NO_WARNINGS".
   Должно выглядить так:
#define _USE_MATH_DEFINES
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <math.h>
